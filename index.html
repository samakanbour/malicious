Are people in Qatar more exposed to cyber attacks than the rest of the world?
In response to this question, we studied 2,000 most visited sites in Qatar and in the world. Using the analytical service providers Alexa Top Sites, Similar Web, Web of Trust, and Virus Total, we were able to extract, classify and study these sites.
Click below to view our findings expressed in a visual storytelling.

The Story

interest

According to Alexa's ranking and SimilarWeb's categorization, Qatar's population is more interested in politics, beauty and health than the rest of the world. Alexa ranking is calculated based on the reach and page views recorded from users who have the Alexa toolbar. The reach refers to the number of unique Alexa users who visit a site in one day. Page views is the number of times a site is visited over a period of 3 months. If a user visits the same URL multiple times on the same day, these visits will be counted as one. SimilarWeb categorization uses a learning algorithm to generate a category for a given website using content tags, similarity results and a learning set of 2.5M categorized websites.

$$category \ rank = {\sum site \ rank \over count \ of \ sites \ in \ category}$$


How to read this graph the higher the category, the more interested is the user community in this category.
Qatar and World categories are linked to help you track the change in interest.

reputation

According to Web of Trust's reputation system, websites browsed in Qatar are less trusted than those browsed in the rest of the world. Reputations are based on ratings received from users and third-party sources on how trustworthy a website is. The reputation is further supported by an explanation that has a negative, questionable, neutral, or positive connotation. A negative explanation means the website is subject to cyber attacks. For our analysis, we only consider reputations and explanations when user confidence is greater or equal to 10%, as suggested by Web of Trust.

$$ { reputation \ class \ \% = {count \ of \ sites \ in \ class\qquad \over total \ classified \ sites\qquad} } \qquad \qquad for \ confidence \ge 10% $$


How to read these charts the first chart shows site reputations in Qatar and in the world, the second highlights the explanation behind the suggested reputations.
